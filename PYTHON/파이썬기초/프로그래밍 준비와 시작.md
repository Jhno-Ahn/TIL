# 파이썬 프로그래밍 준비와 시작

- IDLE 프로그램 설치
    - 파이썬에서 제공하는 기본 개발 툴
    - [https://www.python.org/](https://www.python.org/)

- 설치 후에 IDLE 프로그램을 실행하면 아래와 같은 창이 켜진다.
    
    ![스크린샷 2022-04-27 오후 5.46.34.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/7135bfbb-5acd-4ae8-9068-142c871da40a/스크린샷_2022-04-27_오후_5.46.34.png)
    

- 간단한 실습
    1. IDLE 에디터에서 다음의 내용을 입력한 뒤에
        
        ![스크린샷 2022-04-27 오후 5.51.52.png](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/1915fc60-b308-474d-a4f0-ceae48d6f5fd/스크린샷_2022-04-27_오후_5.51.52.png)
        
    2. F5를 눌러 실행시키면 다음과 같은 결과를 볼 수 있다.
