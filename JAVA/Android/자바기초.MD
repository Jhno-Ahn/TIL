## 2. Casting(타입변환)

### Casting 이란?

- 자바에서 캐스팅은 타입을 변환하는 것이다.
- 상속 관계에 있는 부모와 자식 클래스 간에는 서로 형변환이 가능하다.
- 업 캐스팅
    - 자식 클래스가 부모 클래스 타입으로 캐스팅 되는 것.
- 다운 캐스팅
    - 부모 클래스가 자식 클래스 타입으로 캐스팅 되는 것.
